# Google ADK Examples

A set of example agents written with Google ADK.

- `dice_agent`: Minimal agent example that rolls DnD dice.
- `spell_agent`: A more complex tool-calling agent that looks up DnD spells.
- `wizard_agent`: A complex stateful multi-step multi-agent that "configures" a Level 1 DnD Wizard.

## To Run

```
uv run adk web
```
